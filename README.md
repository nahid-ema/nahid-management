# Membership Dashboard

A modern, responsive, and beautiful Membership Dashboard featuring a clean aesthetic with a deep indigo/violet theme.

## Features
- **Modern UI**: Styled with premium glassmorphism, floating shadows, and vibrant gradients.
- **Responsive**: Fully responsive design for both desktop and mobile views.
- **Dark & Light Mode**: Built-in toggle for stunning dark and light themes.
- **Single File Build**: All CSS, JS, and HTML are bundled into a single robust file.

## Getting Started
To view the project, simply clone the repository and open `index.html` in your web browser.

```bash
git clone <your-repository-url>
cd HTML
# Open index.html in your browser
```
